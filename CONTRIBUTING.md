# Contributing

Contributions should preserve traceability between the Simulink model,
exported datasets, figures, and interactive visualisations.

Before opening a pull request:

1. State which model equations or parameters were changed.
2. Export a new baseline comparison.
3. Update the relevant CSV files and figures.
4. Document the MATLAB and Python versions used.
5. Do not commit generated `slprj`, cache, code-generation, or virtual
   environment folders.
6. Do not present simulation outputs as experimental measurements.
